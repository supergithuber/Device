## Device


**Easy to recognize your device type or model**

#### get the device model

```
Device.model()
```
containing all kinds of iPhone, iPad, iPod, simulator


#### get the device type

```
Device.type()
```

recognize current device, it's iPhone, iPad, iPod... etc


also 

```
Device.isiPad()
Device.isiPod()
Device.isiPhone()
Device.isSimulator()
Device.isRetina()
```
